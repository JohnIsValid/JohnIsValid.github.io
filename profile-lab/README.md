# Profile Lab

这是一个独立的新个人主页实验，不会影响根目录现有主页。

当前版本参考了你给的紫色科幻长页：深色纹理背景、顶部导航、三角主视觉、六个 feature、重叠画廊、观点区、数字区和底部联系面板。

## 文件

- `index.html`：页面内容、分区结构、图片替换位置。
- `styles.css`：紫色科幻视觉系统、响应式布局、几何装饰。
- `script.js`：滚动时给顶部导航加状态。

## 推荐替换的素材

- `../assets/myIcon.jpg`：现在被用作 hero 主视觉和 OUT OF THE BOX 装饰。最好换成透明 PNG 人物、项目角色、游戏物件或更像参考图宇航员的主视觉。
- `../assets/medieval-world-map-bg.png`：画廊左图，适合放代表项目截图或世界地图。
- `../assets/hero-writing-desk.png`：画廊右图和联系区图片，适合放工作台、项目封面或概念图。

## 主要配置

在 `styles.css` 顶部的 `:root` 改：

- `--space-1` / `--space-2`：背景深浅。
- `--violet`：主紫色。
- `--pink`：粉色霓虹强调。
- `--text` / `--muted`：文字颜色。

## 内容替换建议

这个版本的文案只是骨架。你可以重点替换：

- `DISCOVERY`：换成你的主页主标题，比如 `DESIGN ORBIT`、`VALID ARCHIVE`、`WORLD CELL`。
- 六个 feature：换成你的真实能力、项目方向或栏目。
- `WHAT WE THINK ?`：换成你的设计宣言或文章摘要。
- 数字区：换成真实项目数、文章数、原型数，或者直接删掉。
- 联系区：换成邮箱、GitHub、Bilibili、知乎、Notion 等。
